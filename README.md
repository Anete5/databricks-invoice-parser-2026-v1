# Databricks PDF Invoice Parser

This repository contains a single Jupyter notebook that demonstrates end-to-end
PDF invoice parsing using Databricks.

The notebook uses **Databricks AI functions** that are **only available inside a
Databricks workspace**.

## Main notebook
- `pdf_invoice_parser_2026_v1.ipynb`

All implementation details, explanations, and examples are documented directly
inside the notebook.

## How to use
1. Clone or import this repository into a Databricks workspace
2. Open `pdf_invoice_parser_2026_v1.ipynb` from **Databricks Repos**
3. Follow the instructions and markdown cells inside the notebook

## Requirements
- Databricks workspace
- Databricks Runtime with AI functions enabled
- Required permissions to use Databricks AI features

## Notes
This repository is intended as a reference and example implementation.
It will **not run correctly outside Databricks**.
